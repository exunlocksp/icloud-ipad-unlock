Chúng tôi hỗ trợ gì?

iPad WiFi Only

Chỉ nhận iPad bản WiFi. Vui lòng kiểm tra model trên thân máy hoặc trong Cài đặt trước khi liên hệ.

🔓

Mở khóa iCloud

Quy trình

Các bước thực hiện

Làm theo 5 bước dưới đây để gửi yêu cầu mở khóa iCloud cho iPad WiFi của bạn.

01

Restore lên iOS mới nhất

Kết nối iPad với máy tính, dùng iTunes hoặc Finder để Restore iPad lên phiên bản iOS mới nhất. Đảm bảo máy ở màn hình kích hoạt (Hello screen).

02

Dùng Tool đọc ra File

Tải Tool bên dưới, cài đặt trên máy tính rồi dùng để đọc và xuất file từ iPad. Liên hệ Zalo nếu bạn cần hỗ trợ cài đặt.

Tải Tool đọc File (MEGA)
03

Đặt tên File theo Serial Number

Đổi tên file vừa đọc được thành Serial Number của iPad. Serial Number tra được trên thân máy hoặc hộp đựng.

04

Gửi file qua Zalo

Gửi file đã đặt tên qua Zalo số 0967 609 909. Kèm theo ảnh Serial Number nếu có thể.

05

Thanh toán & nhận kết quả

Chuyển khoản theo thông tin bên dưới. Sau khi xác nhận thanh toán, chúng tôi tiến hành xử lý và thông báo kết quả qua Zalo.
