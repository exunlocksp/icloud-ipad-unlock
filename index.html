Quy trình

Các bước thực hiện

Tải Tool → bấm Identify Device → gửi Serial Number qua Zalo & thanh toán. File tự động gửi lên server.

01

Tải Tool spFMI

Tải và cài đặt Tool spFMI trên máy tính Windows. Kết nối iPad với máy tính bằng cáp USB.

Tải Tool spFMI (MEGA)
02

Bấm Identify Device

Mở Tool, chọn iPad trong danh sách rồi bấm nút Identify Device. File sẽ tự động gửi lên server — bạn không cần gửi file thủ công.

☁️

Sau khi bấm Identify Device, ghi lại Serial Number ở cột Serial trong Tool để gửi Zalo ở bước tiếp theo.

03

Gửi Serial Number qua Zalo

Nhắn tin Serial Number của iPad qua Zalo số 0967 609 909 để xác nhận đơn hàng.

Ví dụ Serial: DMPZ65LSKD8C
Gửi Serial qua Zalo
04

Thanh toán & nhận kết quả

Chuyển khoản theo bảng giá và thông tin bên dưới. Sau khi xác nhận thanh toán, chúng tôi tiến hành xử lý và thông báo kết quả qua Zalo.
